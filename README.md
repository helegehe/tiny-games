# 《别点我》微信小游戏（单机版）

这是一个“反向选择”玩法的微信小游戏：
- 屏幕上方给出条件或题目
- 玩家需要点击 **不符合条件**（或知识题中的错误项）
- 主要目标是快速、稳定地做出反向判断

当前仓库为 **纯单机版本**，不依赖后端存储。

---

## 1. 当前已实现功能

### 1.1 核心玩法
- 统一规则：`规则：点不符合条件的项`
- 题目类型：
  - 规则生成题：颜色、类别、形状、数字比较、复合条件
  - 固定知识题：图片识别、科普、历史、地理、诗歌（按难度解锁）
- 每题可点击正确项数量被约束为 `1-2` 个

### 1.2 游戏模式
- 经典闯关
  - 共 20 关
  - 每关 5 题
  - 3 次失误机会
  - 关卡通关有金币奖励
- 限时挑战
  - 整局固定 60 秒
  - 答对自动下一题
  - 答错或时间到立即结束
  - 结算金币 = 本局答对题数
- 每日挑战
  - 使用日期种子生成固定题序
  - 3 次失误机会
  - 完成 2 关即结算

### 1.3 道具系统（广告兑换 + 对局使用）
- 道具工坊：看激励广告兑换库存（+1）
- 道具类型：
  - 排除卡：排除一个错误选项
  - 冻结时间：限时挑战冻结总计时 3 秒
  - 护盾：免疫一次答错
  - 挑剔卡：跳过当前题并直接下一题
- 对局内可直接点道具按钮使用

### 1.4 失败复活
- 失败时弹框询问是否看广告复活
- 完整观看激励广告后可复活继续
- 单局仅允许复活一次

### 1.5 功能中心
- 每日任务（完成/领取）
- 图鉴系统（发现统计）
- 主题包（金币或广告解锁）
- 本地排行榜（限时榜 Top10）
- 道具工坊
- 本地挑战记录（原“留言挑战”，本地记录与处理）

### 1.6 广告接入
- 已封装广告管理器，支持：
  - Banner
  - 激励视频
  - 插屏
- 非微信环境自动降级，不阻塞本地调试
- 需自行填写广告位 ID（见 `src/ads.js`）

### 1.7 隐私说明
- 新增隐私说明页面（首页入口）
- 完整文本见 `PRIVACY.md`

## 2. 不再包含的功能

## 3. 数据存储说明
- 金币、分数、道具、主题、任务、图鉴等均存储在本地
- 微信环境使用 `wx.getStorageSync / wx.setStorageSync`
- 卸载小程序、清缓存或更换设备后，本地数据可能丢失

---

## 4. 运行方式
1. 使用微信开发者工具导入项目根目录
2. 直接运行可体验单机完整流程
3. 若需接入广告，请在 `src/ads.js` 中填写广告位 ID：
   - `banner`
   - `rewarded`
   - `interstitial`

---

## 5. 关键文件
- 入口与配置
  - `game.js`
  - `game.json`
  - `project.config.json`
- 核心源码
  - `src/main.js`：游戏主流程、场景、UI、道具/复活逻辑
  - `src/logic.js`：出题与判定（反向选择核心）
  - `src/data.js`：题库、难度、配置常量
  - `src/storage.js`：本地存档
  - `src/ads.js`：广告封装（微信 API + 降级）
  - `src/tasks.js`：每日任务逻辑
  - `src/social.js`：本地挑战记录
  - `src/audio.js`：音效管理
  - `src/assets.js`：资源加载
- 文档
  - `PRIVACY.md`：隐私保护指引
